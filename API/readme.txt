Option 3
https://petstore.swagger.io/
- Create a user
- Find the created user
- Update the user's name and email
- Find the updated user
- Delete the user

Folder structure:
- petstore.cy.js contains all the actual test execution
- petstore.json is a fixture created to store users data

How to run:
1. Clone the DEVSU repository from github to you local machine
2. Install dependencies with 'npm install'
3. Install cypress with 'npm install cypress --save-dev'
4. Execute the test running 'npx cypress open'
